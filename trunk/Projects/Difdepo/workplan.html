Séminaire ALGORITM: session 2 Transcription des comptes rendus de l'Oulipo

Seminar objectives
  • decide on a markup scheme for a subset of the Oulipo archive
  • experiment with different ways of marking up some samples
  • demonstrate rendering and analysis of marked up samples
Working files
These need to be prepared...
  • we have about 200 or so page images relating to meeting minutes; a selection of maybe a dozen needs to be made for working purposes
  • we make available all page images, plus plain text transcriptions for the selected dozen, plus a few marked up according to a preliminary schema
  • we prepare in advance a simple ms-transcription schema and write an ODD for it; if time permits we can have a session on how to modify this schema if it proves inadequate during discussion.
Proposed timetable
  1. 0930 Introductions, explanations etc. (15 mins)
  2. 0945 Investigation of the data. Students look at page images and propose things that need to be captured (30 mins)
  3. 1015 Presentation and validation of preliminary schema (SR) : look at sample file in Oxygen (45 mins) (LB)
  4. 11h: Coffee break (30 mins)
  5. Students divide into three groups :
    • Group 1 : Working in pairs, they transcribe a document in Word, improve its styles, export it to TEI, and check the markup using Oxygen (may need customised XSLT transform?)
    • Group 2 : Working in pairs they choose a doc, experiment with Fromthepage ; export it as TEI ; check/enhance markup using Oxygen
    (60 mins)
  6. 12h30 Lunch (90 mins)
  7. 14h Practicals continue: students swop over around and try different approach (60 mins)
  8. 15h30 Demonstrate functioning web apps able to do some of the following:
    • display graphic and transcription in parallel
    • visualise attendance records of oulipiens over time
    • word lists and co-occurence lists over time -
    • display annotated versions of text
    (60 mins)
  9. 16h30 Review and discussion (30 mins)
