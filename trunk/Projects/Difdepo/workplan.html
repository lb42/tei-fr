Séminaire ALGORITM: session 2 Transcription des comptes rendus de l'Oulipo

Seminar objectives
  • decide on a markup scheme for a subset of the Oulipo archive
  • experiment with different ways of marking up some samples
  • demonstrate rendering and analysis of marked up samples
Working files
These need to be prepared...
  • we have about 200 or so page images relating to meeting minutes; a selection of maybe a dozen needs to be made for working purposes
  • we make available all page images, plus plain text transcriptions for the selected dozen, plus a few marked up according to a preliminary schema
  • if we had more than one day we could develop a schema before their very eyes; as we don't I propose to prepare in advance a simple ms-transcription schema and write an ODD for it; if time permits we can have a session on how to modify this schema if it proves inadequate during discussion.
Proposed timetable
  1. Introductions, explanations etc. (15 mins)
  2. Investigation of the data. Students look at page images and propose things that need to be captured (30 mins)
  3. Presentation and validation of preliminary schema: look at a sample file in Oxygen (40 mins)
  4. 11h: Coffee break (30 mins)
  5. Students divide into three groups :
    • Practical 1 : some students (working in pairs) mark up a document using Oxygen.
    • Practical 2 : other students mark up document in Word using predefined styles, generate TEI from it (using predefined stylesheet), validate/correct
    • Practicals 3 : yet other students experiment with web-based cooperative editing tool?
    (60 mins)
  6. 12h30 Lunch (90 mins)
  7. Practicals continue: all students move around and try different approach (30 mins)
  8. Demonstrate functioning web apps able to do some of the following:
    • display graphic and transcription in parallel
    • visualise attendance records of oulipiens over time
    • word lists and co-occurence lists over time -
    • display annotated versions of text
    (60 mins)
  9. Review and discussion (30 mins)
